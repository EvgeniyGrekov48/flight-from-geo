# Описание

Flight From Geo - Карта парапланерных объектов
Веб-приложение на Angular 20 для интерактивного отображения тематических объектов на карте (горы для парапланеризма, термические потоки, аэродромы).

Прототип системы автоматической обработки геоданных.
## Запуск приложения

Для запуска приложения при разработке, выполните:

```bash
ng serve
```

Приложение будет запущено в праузере по адресу `http://localhost:4200/`.
## Запуск имитации сервера (зглушка)

Имитация сервера реализовани с помощью библиотеки `json-server`, для запуска выполните команду:

```bash
json-server --watch ./json-server/db.json --host  0.0.0.0
```

Сервер запустится на `http://localhost:3000`
